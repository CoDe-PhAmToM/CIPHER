We Decode
Hard Problems.

CIPHER nació del primer equipo ICPC de la UPEA. Vimos que nuestra universidad no tenía una comunidad de programación — así que la construimos nosotros. Abierta a todos. Impulsada por la curiosidad, los algoritmos, y la satisfacción de resolver lo que parecía imposible.

// 002 — Activities

What We Do

// 01

Weekly Contests

Rondas cronometradas cada semana en Codeforces o VJUDGE. Editorial completo + debrief post-ronda. El hábito es la clave.

// 02

Algorithm Workshops

Sesiones profundas sobre un tópico por semana: segment trees, DP, grafos, geometría. Presentados por miembros para miembros.

// 03

Team Formation

Formamos tríos balanceados para competencias. Todos tienen equipo. La dinámica de trabajo en equipo se aprende practicándola.

// 04

Apoyo Académico

Programación I/II, POO, Estructura de Datos, BD I/II — resolvemos dudas entre pares antes de que se conviertan en deudas.

// 05

Study Groups

Grupos de 3-4 personas atacando un problem set juntos. La mejor forma de aprender es explicar lo que ya sabes.

// 06

UPEA Hackathon

Nuestra competición interna abierta a toda la universidad. Sembramos la cultura de programación competitiva en la UPEA.

Fundamentos

Aprende a Programar Bien

Si estás comenzando o quieres reforzar las bases de la carrera, este es tu punto de partida. Cubrimos exactamente lo que se lleva en Ing. Sistemas — con orientación práctica y aplicada.

Programación I — Lógica y sintaxis
Programación II — Funciones y módulos
POO — Clases, herencia, polimorfismo
Estructura de Datos — Listas, árboles, grafos
Base de Datos I — Modelado relacional, SQL
Base de Datos II — SQL avanzado, NoSQL
Algoritmos

Pensamiento Algorítmico

Ya dominas la programación. Ahora entrenamos la parte más importante: cómo analizar un problema, diseñar una solución eficiente y razonar sobre su complejidad. Esto es lo que diferencia a un programador de un resolvedor de problemas.

Análisis de complejidad — Big O
Búsqueda y Ordenamiento avanzado
Recursión y Divide & Conquer
Programación Dinámica (DP)
Grafos — BFS, DFS, Dijkstra, Floyd
Greedy Algorithms & correctness proofs
Solver Pro

Competencia de Verdad

Para quienes apuntan a contests serios y al ICPC. Tópicos avanzados, presión de tiempo real, análisis post-contest. Aquí se separan los que resuelven de los que solo intentan.

Segment Trees & Fenwick Trees
DP Avanzado — Bitmask, Knapsack, SOS
Network Flow & Matching
Geometría Computacional
String Algorithms — KMP, Suffix Array
Mock Contests ICPC-style (5h, equipo)
